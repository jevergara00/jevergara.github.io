HTML 5 es un Lenguaje de Marcado de Hipertexto (HyperText Markup Language) que funciona con las bases del HTML de siempre, pero que cuenta con etiquetas adicionales que permiten crear sitios webs más modernos y más compatibles con las tecnologías digitales actuales. HTML no es un lenguaje de programación como tal, sino una manera de estructurar, interpretar y desplegar el contenido de un documento web
